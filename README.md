PLANORY – Virtual Expense Tracker with Smart Budget Insights
Track expenses, analyze spending patterns, and receive intelligent budget recommendations.

What It Does:

Accurately records and categorizes daily expenses.
Provides AI-based budget recommendations based on spending behavior.
Generates analytics using charts for better financial insights.
Supports goal-based savings tracking.
Stores data locally using Room Database and syncs with Firebase.

REQUIREMENTS :

Kotlin
Android Studio
Room Database
Firebase Firestore
MPAndroidChart
Coroutines

How this App works :

User logs expenses with category and amount.
The app stores data locally and syncs it with cloud.
Analytics dashboard displays spending patterns using charts.
AI module analyzes spending and provides budget suggestions.
User can set financial goals and track progress over time.

Purpose :

This application helps users manage their finances effectively by providing insights, analytics, and smart budget recommendations.

Compilation Steps :

Install Android Studio.
Open the project.
Sync Gradle.
Run the application on emulator or device.
